What are Event Listeners?
Instead of writing events inside HTML (onclick="..."),
we attach them with JavaScript code using addEventListener.
This keeps HTML clean and allows multiple functions on the same event.
✅ Syntax
element.addEventListener("event", functionName);
element → The DOM element (e.g., button, div, input).
event → Event type (click, mouseover, keydown, etc).
functionName → Function that will run when the event occurs.
