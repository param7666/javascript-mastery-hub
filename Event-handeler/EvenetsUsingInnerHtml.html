Demo on Events Using Inline HTML Attribute

1) onclick

When it fires: When the user clicks (press + release) the left mouse button on an element.

2) ondblclick

When it fires: When the user quickly clicks twice on the same element.

Double Click me to change color

3) onmouseover

When it fires: When the mouse pointer enters the element area.

Hover Me

4) onmouseout

When it fires: When the mouse pointer leaves the element area.

Hover Me

5) onmousedown

When it fires: When the mouse button is pressed down on an element.

6) onmouseup

When it fires: When the mouse button is released over an element.

7) onmousemove

When it fires: When the mouse is moved over the element.

8) oncontextmenu

When it fires: When the user right-clicks on an element.
