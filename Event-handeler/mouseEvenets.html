Mouse Events
  • Mouse events happen when you click, hover, or interact with the mouse.
  • Common Mouse Events:
  • click → when you click on an element
  • dblclick → double-click
  • mouseover → mouse pointer enters
  • mouseout → mouse pointer leaves
  • mousedown → mouse button pressed
  • mouseup → mouse button released
  • mousemove → when you move the mouse
